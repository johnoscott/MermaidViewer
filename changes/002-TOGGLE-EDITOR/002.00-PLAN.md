# 002 - Toggle Editor Panel

## Summary

Hide the editor panel by default when opening `.mmd` files in MermaidViewer. Add a toggle button in the preview toolbar to show/hide the editor.

## Changes

### ContentView.swift

- Added `@State private var showEditor: Bool = false` â€” editor hidden by default
- Wrapped editor `VStack` in `if showEditor { ... }` for conditional rendering
- Added sidebar toggle button (`sidebar.left` SF Symbol) at the leading edge of the preview toolbar
- Dynamic `minWidth`: 800px when editor is shown, 400px when hidden
- Toggle animates with `withAnimation`

## Behaviour

- App launch: editor hidden, shows diagram preview only
- Toggle button in toolbar shows/hides editor
- State is session-only (not persisted to UserDefaults)
